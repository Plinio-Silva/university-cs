# 🧪 Laboratório de Programação

Disciplina prática focada na implementação de algoritmos e no desenvolvimento de habilidades de programação.

## 📚 Conteúdos principais
- Implementação dos algoritmos estudados em Introdução à Programação  
- Entrada e saída de dados  
- Estruturas condicionais e de repetição  
- Funções e modularização  
- Exercícios práticos em laboratório  
- Resolução de problemas reais em código  

## 📂 Estrutura
- `exercicios/`
- `projetos/`
- `exemplos/`

## 🎯 Objetivo
Aplicar na prática os conceitos fundamentais de programação, desenvolvendo lógica, clareza e organização no código.
