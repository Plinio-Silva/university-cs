# 📚 1º Período — Ciência da Computação

Este diretório reúne os projetos, exercícios e materiais desenvolvidos durante o **1º período** da graduação em Ciência da Computação.  
O foco principal deste semestre é a construção de bases sólidas em lógica, algoritmos e programação.

---

## 🧠 Disciplinas do 1º Período

### 🔹 Introdução à Programação
Disciplina teórica que apresenta os conceitos fundamentais de programação e resolução de problemas.

**Conteúdos principais:**
- Variáveis e tipos de dados  
- Operadores aritméticos e lógicos  
- Estruturas condicionais  
- Estruturas de repetição  
- Funções e modularização  
- Algoritmos em pseudocódigo e/ou Portugol  

---

### 🔹 Laboratório de Programação
Disciplina prática focada em implementação de algoritmos em uma linguagem de programação (por exemplo, **C** ou **Python**).

**Conteúdos principais:**
- Implementação dos algoritmos vistos em Introdução à Programação  
- Manipulação de entrada e saída  
- Resolução de problemas em código  
- Boas práticas iniciais de programação  
- Exercícios em laboratório


---

## 🎯 Objetivos do 1º Período
- Desenvolver raciocínio lógico  
- Aprender a transformar problemas em algoritmos  
- Implementar soluções em código  
- Criar base sólida para disciplinas futuras (Estruturas de Dados, POO, etc.)

---

## 🛠️ Tecnologias Utilizadas
- **C** e/ou **Python**  
- VS Code / GDB ONLINE  
- Terminal / Git Bash
